# 🎮 Game Commerce

A modern, high-converting e-commerce web application focused on gaming products such as PlayStation, Xbox consoles, and accessories.

---

## 🚀 Overview

Game Commerce is a responsive and visually engaging e-commerce platform designed for gamers. It provides a smooth shopping experience with a clean UI, fast navigation, and a functional cart system.

---

## ✨ Features

* 🛒 Add to cart / remove products
* 🔢 Quantity management
* 🧾 Checkout UI
* 📱 Fully responsive design
* 🎮 Gaming-focused UI (dark theme + neon accents)
* ⚡ Fast performance with Vite
* 🧩 Component-based architecture

---

## 🧱 Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **State Management:** React Hooks
* **Data:** Mock data (local)

## 🛠️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/game-commerce.git
cd game-commerce
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open in browser:

```
http://localhost:5173
```
